# Chrome Web Store 资源文件

## 截图要求

### 主截图 (必需)
- **尺寸**: 1280x800 或 640x400 像素
- **格式**: PNG 或 JPEG
- **用途**: 商店列表页面的主要展示图
- **建议**: 展示插件主界面，突出核心功能

### 其他截图 (最多5张)
- **尺寸**: 1280x800 或 640x400 像素
- **格式**: PNG 或 JPEG
- **用途**: 详细功能展示

## 图标要求

### 扩展图标
- **16x16**: 用于扩展栏
- **48x48**: 用于扩展管理页面
- **128x128**: 用于Chrome Web Store

### 商店图标
- **128x128**: 用于商店列表
- **440x280**: 用于商店详情页横幅

## 使用说明

1. 将现有截图复制到此目录
2. 重命名为标准格式
3. 确保尺寸符合要求
4. 上传到Chrome Web Store
